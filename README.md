# Academic Organization Ontology

## Overview
This project presents an Academic Organization Ontology developed in Protégé using OWL 2. The ontology models the structure and relationships of an academic institution including students, professors, courses, departments, classrooms, examinations, and academic programs.

## Features
- Developed using Protégé
- Implemented in OWL 2
- Logical consistency verified using HermiT Reasoner
- Includes classes, object properties, data properties, and individuals
- Supports DL Queries for ontology reasoning
